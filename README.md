Beetnik
=======

Test out the power of beat detection algorithms using Web Audio!

Runs on Meteor because that makes life less depressing.

Setup
-----

Install

- Meteor
- Meteorite

or Arch Linux users can use a script that does all this

    ./scripts/setup.zsh

Running Beetnik
---------------

    cd mrt

or can run the script

    ./scripts/mrt.zsh
